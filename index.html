Server Commands

Daftar lengkap command yang tersedia di RimbaPixel

Economy & Shop

/rshop
Membuka menu toko server untuk membeli dan menjual item
/bal
Cek saldo money kamu saat ini
/pay [player] [amount]
Transfer uang ke player lain
/sell hand
Jual item yang sedang kamu pegang
/ah
Buka auction house untuk jual beli dengan player lain

Warp & Teleport

/warp
Lihat daftar semua warp yang tersedia
/warp [nama]
Teleport ke warp tertentu (spawn, mall, farm, dll)
/pvpwarp
Teleport ke arena PvP untuk bertarung
/spawn
Kembali ke spawn point utama
/home
Teleport ke home yang sudah kamu set
/sethome [nama]
Set home baru di lokasi kamu berdiri

Claim & Protection

/claim
Claim area untuk melindungi build kamu
/claimlist
Lihat daftar semua claim yang kamu miliki
/trust [player]
Beri akses ke player lain di claim kamu
/untrust [player]
Hapus akses player dari claim kamu
/abandonclaim
Hapus claim yang sedang kamu berdiri di dalamnya

Info & Rules

/rules
Lihat semua aturan server RimbaPixel
/info
Informasi umum tentang server
/help
Bantuan dan panduan bermain
/discord
Dapatkan link Discord server
/vote
Link untuk vote server dan dapatkan reward

Social & Party

/msg [player] [pesan]
Kirim pesan private ke player lain
/party create
Buat party baru untuk bermain bersama
/party invite [player]
Undang player ke party kamu
/ignore [player]
Blokir chat dari player tertentu
/tpa [player]
Minta teleport ke player lain
/tpaccept

Utility & Tools

/kit [nama]
Ambil kit starter atau daily kit
/suicide

/nick [nama]
Ubah nickname kamu di server

Server Rules

Aturan yang harus dipatuhi untuk menjaga kenyamanan bermain bersama

1. No Griefing & Stealing - Dilarang merusak build player lain atau mengambil item tanpa izin
2. No Cheating/Hacking - Penggunaan hack, cheat, atau exploit akan mendapat ban permanen
3. Respect Others - Hormati semua player, no toxic behavior, spam, atau harassment
4. No Advertising - Dilarang promosi server lain atau link yang tidak relevan
5. Use English/Indonesian - Gunakan bahasa Indonesia atau Inggris di chat publik
6. No Excessive PvP - PvP hanya di area yang ditentukan, no random killing
7. Keep it Family Friendly - Jaga konten tetap ramah untuk semua umur
8. No Lag Machines - Dilarang membuat redstone atau build yang menyebabkan lag

Terms of Service

Terakhir diperbarui: 27 Juli 2025

Selamat datang di RimbaPixel! Dengan menggunakan server Minecraft kami, Anda menyetujui untuk mematuhi ketentuan layanan ini.

1. Penerimaan Ketentuan

Dengan mengakses dan menggunakan server RimbaPixel, Anda menyetujui untuk terikat dengan ketentuan layanan ini. Jika Anda tidak setuju dengan ketentuan ini, harap jangan menggunakan layanan kami.

2. Deskripsi Layanan

RimbaPixel adalah server Minecraft Indonesia yang menyediakan:

  • Server Survival RPG dengan fitur custom
  • Sistem ekonomi dan trading
  • Boss fights dan custom items
  • Land protection dan claiming system
  • Cross-platform support (Java & Bedrock)

3. Persyaratan Pengguna

3.1 Usia Minimum

Anda harus berusia minimal 13 tahun untuk menggunakan layanan kami. Jika Anda berusia di bawah 13 tahun, Anda harus mendapatkan izin dari orang tua atau wali.

3.2 Akun Minecraft

  • Anda harus memiliki akun Minecraft yang valid
  • Username harus sesuai dengan aturan server
  • Dilarang menggunakan akun yang dibeli atau diperoleh secara ilegal

4. Aturan Server

Pelanggaran aturan dapat mengakibatkan peringatan, mute, kick, atau ban permanen.

4.1 Larangan Umum

  • No Griefing: Dilarang merusak build player lain
  • No Stealing: Dilarang mengambil item tanpa izin
  • No Cheating: Dilarang menggunakan hack, cheat, atau exploit
  • No Spam: Dilarang spam chat atau command
  • No Advertising: Dilarang promosi server lain
  • No Harassment: Dilarang bullying atau toxic behavior

4.2 Aturan Chat

  • Gunakan bahasa Indonesia atau Inggris
  • Jaga sopan santun dan hormati player lain
  • Dilarang menggunakan kata-kata kasar atau tidak pantas
  • Dilarang spam atau flood chat

4.3 Aturan Gameplay

  • PvP hanya di area yang ditentukan
  • Dilarang membuat lag machine atau redstone yang berlebihan
  • Gunakan fitur claim untuk melindungi build Anda
  • Ikuti petunjuk admin dan moderator

5. Hak Kekayaan Intelektual

  • Semua konten di server adalah milik RimbaPixel
  • Plugin dan sistem custom dilindungi hak cipta
  • Dilarang menyalin atau mendistribusikan konten server
  • Build yang Anda buat tetap menjadi hak Anda

6. Privasi dan Data

Penggunaan data Anda diatur dalam Privacy Policy kami. Dengan menggunakan layanan, Anda menyetujui pengumpulan dan penggunaan data sesuai kebijakan tersebut.

7. Pembatasan Tanggung Jawab

  • Server disediakan "sebagaimana adanya"
  • Kami tidak bertanggung jawab atas kehilangan data atau item
  • Kami tidak menjamin server akan selalu online 24/7
  • Kami berhak mengubah fitur atau aturan tanpa pemberitahuan

8. Penghentian Layanan

Kami berhak menghentikan atau membatasi akses Anda ke server jika:

  • Anda melanggar ketentuan layanan
  • Anda melakukan aktivitas yang merugikan server
  • Untuk pemeliharaan atau perbaikan server
  • Atas permintaan hukum yang berlaku

9. Perubahan Ketentuan

Kami dapat mengubah ketentuan layanan ini dari waktu ke waktu. Perubahan akan diumumkan di website dan akan berlaku efektif setelah diposting. Penggunaan berkelanjutan setelah perubahan berarti Anda menyetujui ketentuan baru.

10. Hukum yang Berlaku

Ketentuan layanan ini diatur oleh hukum Indonesia. Setiap sengketa akan diselesaikan melalui musyawarah atau melalui pengadilan yang berwenang di Indonesia.

11. Kontak

Untuk pertanyaan tentang ketentuan layanan ini, silakan hubungi:

  • Discord: RimbaPixel Server
  • Email: admin@rimbapixel.my.id
  • Website: rimbapixel.my.id

12. Persetujuan

Dengan menggunakan server RimbaPixel, Anda mengakui bahwa Anda telah membaca, memahami, dan menyetujui ketentuan layanan ini.
